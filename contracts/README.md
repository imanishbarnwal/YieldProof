# YieldProof Contracts

Smart contracts for YieldProof, deployed on Mantle Network.

## Setup

1. Copy `.env.example` to `.env` and fill in your private key.
2. Install dependencies:
   ```bash
   npm install
   ```

## Commands

- `npm run compile`: Compile contracts
- `npm run test`: Run tests
- `npm run deploy`: Deploy to Mantle Testnet
