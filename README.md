# YieldProof

YieldProof is a Web3 hackathon MVP build on the Mantle EVM network.

## Structure

- `/contracts`: Smart contracts using Hardhat and Solidity.
- `/frontend`: Web application using Next.js, TypeScript, and Tailwind CSS.
- `/docs`: Pitch notes and demo script.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run frontend:
   ```bash
   npm run frontend
   ```

3. Compile contracts:
   ```bash
   npm run contracts:compile
   ```
